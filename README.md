# Simple Python Chatbot

A basic Command Line Interface (CLI) chatbot written in Python.

## Features
- Responds to basic greetings (hello, hi).
- Answers simple questions (how are you).
- Exits gracefully (bye).

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ragibneaz/chat_bot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chat_bot
   ```

## Usage

Run the chatbot using Python:

```bash
python3 chatbot/main.py
```

## Example Interaction

```
ChatBot: Hello! Type 'bye' to exit.
You: hello
ChatBot: Hello there! How can I help you?
You: bye
ChatBot: Goodbye! Have a nice day!
```
